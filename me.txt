# ANOTAÇÃO DE RACIOCÍNIOS DE FUTURAS FEATURES #


A rodando ONLINE pela VERCEL -> pasta BUILD/index.js
    O NODE_ENV deve ser 'production'
            Acessa o banco de dados oficial do evento


A api rodando LOCALMENTE, posso dar:
    NPM RUN DEV:
        NODE_ENV = 'development'
        Deverá alterar os dados do banco de dados local OU o um  banco de dados separado para DEV

    NPM RUN PROD:
        NODE_ENV = 'production'
        Irá acessar e alterar os dados do banco de dados em produção
